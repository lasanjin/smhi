## Description
Outputs weather forecast data from SMHI api in terminal.


## Install
```
$ curl "https://raw.githubusercontent.com/lasanjin/smhi-cli/master/install.sh" | bash
```

## How to run
```
$ smhi $1
```

- `$1` 
  -  *optional*
  -  forecast (number of days)
  -  input `0-9`, default is today's weather
